# Prophet.AI

An AI-powered destiny calculation engine combining Bazi, Zi Wei Dou Shu, and Western Astrology.

## Features

- **Three Major Systems**: Start with Bazi, switch to Western or Ziwei.
- **AI Analysis**: Uses Google Gemini Pro for deep karmic interpretation.
- **Interactive UI**: Fate River visualization and The Council of Five Elders.

## Deployment

Running on Cloudflare Pages.
