# UI/UX 改版發布：工具庫與統一模板

## 總覽

本次發布將 TaiCalc 從簡單的工具列表轉型為產品級平台。我們推出了具備豐富篩選與排序功能的集中式工具庫 (`/calculators`)，以及一個提供一致、互動且視覺效果驚豔 („Wow”) 的統一計算機模板。

## 關鍵功能

### 1. 工具庫 (`/calculators`)

- **集中式樞紐**: 專屬頁面列出所有 16 個工具。
- **豐富元數據**: 由 `uiCatalog.ts` 驅動，提供一致的分類、標籤與「一句話介紹」。
- **篩選與搜尋**: 客戶端即時篩選，支援依名稱、分類與標籤搜尋。
- **智慧排序**: 支援「精選推薦」(Curated) 與「熱門程度」(Priority) 排序，突顯明星工具。
- **預覽抽屜**: 滑出式抽屜 (`?preview=id`)，讓使用者在開始前先了解「你會得到什麼」。

### 2. 統一計算機模板

- **永遠開啟的輸入區 (Always-Open Input)**: 桌面版 Sticky 側邊欄，確保表單隨時可見、易於修改。
- **互動式結果**: 結果顯示於主內容區域，並帶有流暢的進場動畫。
- **空狀態優化**: 友善的「準備開始」畫面，條列價值主張取代原本的空白畫面。
- **一鍵範例**: 提供「帶入範例資料」按鈕，大幅降低使用者試用的門檻。

### 3. "Wow" 視覺化效果

- **Adapter 模式**: 彈性的 `adapter` 系統，將後端原始 JSON 轉換為標準化的 UI 模型。
- **KPI 卡片**: 重要指標 (如：「每月還款」、「總利息」) 以醒目的大數字卡片呈現於頂部。
- **動態圖表**:
  - `capital.growth`: 資產累積面積圖 (Area Chart)。
  - `tax.calculate`: 稅額 vs 淨收堆疊圖 (Stacked Bar)。
  - `mortgage.calculate`: 本息攤還曲線圖。
  - `salary.analyze`: 年薪結構圓餅圖 (Pie Chart)。
- **詳細區塊**: 結構化的數據表格，用於深度資料展示。

## 技術變更

### 架構

- **`lib/skills/uiCatalog.ts`**: UI 設定的唯一真實來源 (Metadata, 排序權重, 圖表設定)。
- **`lib/adapters`**: 中介層，負責將後端輸出適配為前端視覺化所需的格式。
- **`components/calculators`**: 模組化元件架構 (`SkillCard`, `FilterSidebar`, `CalculatorPageShell`)。

### 可重用 UI 元件

- 實作輕量級、基於 Tailwind 的基礎元件：`Badge`, `Button`, `Card`, `Sheet`。
- 移除 UI 基礎元件的外部依賴，確保穩定性。
