# 中文溝通規範

## 語言使用規範

- **主要語言**: 使用繁體中文進行所有溝通
- **技術術語**: 可適當使用英文技術術語，但需提供中文說明
- **代碼註解**: 使用繁體中文撰寫代碼註解和文檔
- **用戶界面**: 所有用戶可見的文字都使用繁體中文

## 溝通風格

- 保持專業但友善的語調
- 使用清晰簡潔的表達方式
- 適當使用技術術語，但確保易於理解
- 在解釋複雜概念時提供具體範例

## 文檔撰寫

- 規格文檔使用繁體中文撰寫
- 代碼範例包含繁體中文註解
- 錯誤訊息和用戶提示使用繁體中文
- API 文檔提供繁體中文說明
